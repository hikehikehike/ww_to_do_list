## Installation

Python3 must be already installed

```shell
git clone https://github.com/hikehikehike/ww_to_do_list
cd ww_to_do_list
python3 -m venv venv
sourse venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
